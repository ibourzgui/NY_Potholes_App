API

Available Endpoints

Welcome to our API! Follow the 3 easy steps below to get access to most recent data:

NY Pothole Data API
Step 1:

This does a GET to the data.cityofnewyork.us to get pothole data in a json format.

  • /api/v1.0/getdata

Step 2:

This endpoint gets the first 2000 pothole records from the City of New York and inserts it into MongoDB.

  • /api/v1.0/insertdata

Step 3:

This does a query to mongo to retrieve the documents available.

  • /api/v1.0/querydata

Pothole GIF API:

This is a proxy of the Tenor API and returns the first 8 Pothole Pictures

  • /api/v1.0/getgifs
