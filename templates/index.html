Our Mission

The mission of Pothole Data is to plot NY Potholes in an easy to view consumable map. We obtain our data from New York City Open Data (NYC Open Data).

The source is a JSON and is rendered using Flask API. Socrata was used to access the NYC Open Data API and let us more easily manipulate the data. The live data is then stored in a Mongo Database and is manipulated using the Javascript (JS) library d3 and visualized using another JS library Leaflet.

Our visualizations include several different views. The visualizationa can be toggled to view the "status" (open or closed) of the reported pothole. Our visualizations also allow the user to filter the data by borough, zipcode, date reported, etc.
